# ClassHub

ClassHub is a centralized classroom management web application designed to eliminate the communication chaos often caused by scattered messaging platforms like WhatsApp.

It provides a structured digital environment where teachers and students can interact efficiently through a single platform.

## Features

* Teachers can post announcements instantly
* Manage and distribute assignments
* Update class timetables
* Track student attendance
* Maintain organized classroom communication

## Student Benefits

* Access announcements in one place
* View assignments and deadlines easily
* Check updated timetables
* Monitor attendance records through a clean dashboard

## Technology Used

* HTML
* CSS
* JavaScript
* MERN Stack (MongoDB, Express.js, React.js, Node.js)
